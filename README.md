# association-mining-rule
Market Basket Analysis using Python and MLxtend

This project implements association rule mining on an online retail dataset using the Apriori algorithm to uncover meaningful patterns in customer purchasing behavior. The analysis aims to identify product combinations frequently bought together.

Dataset

The dataset is a transactional record of an online retail store, containing 525,461 rows and 8 columns:

Invoice: Transaction ID
StockCode: Product ID
Description: Product Name
Quantity: Quantity purchased
InvoiceDate: Date of purchase
Price: Unit price
Customer ID: Unique customer ID
Country: Country of origin
